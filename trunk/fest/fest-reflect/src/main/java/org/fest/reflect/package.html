Provides a Fluent Interface for the Java Reflection API.

Here are some examples: 

   // import static {@link org.fest.reflect.Reflection org.fest.reflect.Reflection}.*;

   Person person = {@link org.fest.reflect.Reflection#constructor constructor}().withParameterTypes(String.class)
                                .in(Person.class)
                                .newInstance("Yoda");
   {@link org.fest.reflect.Reflection#method(String) method}("setName").withParameterTypes(String.class)
                    .in(person)
                    .invoke("Luke");
   {@link org.fest.reflect.Reflection#field(String) field}("name").ofType(String.class)
                .in(person)
                .set("Anakin");
